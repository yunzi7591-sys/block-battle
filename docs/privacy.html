プライバシーポリシー

最終更新日: 2026年3月15日

Block Battle（以下「本アプリ」）の運営者（以下「当社」）は、ユーザーのプライバシーを尊重し、個人情報の保護に努めます。本プライバシーポリシーは、本アプリが収集する情報、その利用目的、およびユーザーの権利について説明します。

1. 収集する情報

本アプリは、サービスの提供に必要な最小限の情報のみを収集します。

データ項目 収集方法 利用目的
匿名ユーザーID Firebase Auth自動生成 ユーザー識別・データ管理
プレイヤー名 ユーザー入力 対戦画面・ランキング表示
ゲームスコア ゲームプレイ ランキング・自己記録
PvPレーティング 対戦結果から自動算出 マッチメイキング・ランキング

収集しない情報

本アプリは以下の情報を一切収集しません。

  • メールアドレス、電話番号、氏名等の個人を特定できる情報
  • 位置情報
  • デバイスの連絡先、写真、カメラ等へのアクセス
  • 広告識別子（IDFA / AAID）
  • Cookieやトラッキングデータ

2. 情報の利用目的

収集した情報は、以下の目的のみに使用します。

  1. ゲームサービスの提供・運営（対戦マッチング、スコア管理、ランキング表示）
  2. サービスの改善・不具合の修正
  3. 不正行為の検出・防止

3. 情報の保管

  • 保管場所: Google Firebase（Cloud Firestore および Realtime Database）上に保管されます。データはGoogleのセキュリティ基準に基づいて保護されています。
  • 保管期間: ユーザーがアカウントを削除するまで保管されます。アカウント削除後は速やかにすべてのデータが削除されます。

4. 第三者への提供

当社は、以下の場合を除き、ユーザーの情報を第三者に提供しません。

  • 法令に基づき開示が求められた場合
  • ユーザーの生命、身体または財産の保護のために必要がある場合

なお、本アプリはサードパーティの広告ネットワークやアナリティクスサービスを使用していません。

5. ユーザーの権利

ユーザーは以下の権利を有します。

アカウントの削除

アプリ内の「プロフィール」画面から、いつでもアカウントとすべての関連データを削除できます。削除されたデータは復元できません。

データのエクスポート

ご自身のデータの開示を希望される場合は、下記のお問い合わせ先までご連絡ください。

6. 児童のプライバシー

本アプリは13歳未満の児童を対象としたものではありません。13歳未満の児童の個人情報を意図的に収集することはありません。13歳未満のお子様が本アプリを利用する場合は、保護者の監督のもとで行ってください。

7. セキュリティ

当社は、Firebase Security Rulesによるアクセス制御、データの暗号化（通信時のTLS暗号化を含む）、匿名認証によるプライバシー保護など、適切なセキュリティ対策を講じています。

8. プライバシーポリシーの変更

当社は、本プライバシーポリシーを随時更新する場合があります。重要な変更がある場合は、本ページ上で通知します。最新のポリシーは常にこのページでご確認いただけます。
