プライバシーポリシー

最終更新日: 2026年3月19日

Block Battle（以下「本アプリ」）の運営者（以下「当社」）は、ユーザーのプライバシーを尊重し、個人情報の保護に努めます。本プライバシーポリシーは、本アプリが収集する情報、その利用目的、およびユーザーの権利について説明します。

1. 収集する情報

本アプリは、サービスの提供に必要な最小限の情報のみを収集します。

データ項目 収集方法 利用目的
匿名ユーザーID Firebase Auth自動生成 ユーザー識別・データ管理
プレイヤー名 ユーザー入力 対戦画面・ランキング表示
ゲームスコア ゲームプレイ ランキング・自己記録
PvPレーティング 対戦結果から自動算出 マッチメイキング・ランキング

2. 広告について

本アプリは、Google AdMob を利用して広告を配信しています。AdMob は、広告の配信および効果測定のために、以下の情報を取得する場合があります。

  • デバイスの広告識別子（IDFA）: iOS の「App Tracking Transparency」機能により、ユーザーが許可した場合にのみ取得されます。ユーザーが追跡を許可した場合、パーソナライズ広告の配信に使用されます。
  • 広告の表示回数・クリック等のインタラクションデータ
  • デバイスの種類、OS バージョン等の一般的な技術情報

ユーザーが追跡を許可しなかった場合、パーソナライズされない一般的な広告が表示されます。広告に関するデータの取り扱いについては、Google のプライバシーポリシーをご参照ください。

Google Firebase Analytics について

AdMob SDK には Google Firebase Analytics の機能が含まれており、広告の効果測定のために匿名化されたアプリ使用状況データ（イベントデータ、セッション情報等）が自動的に収集される場合があります。このデータは個人を特定するものではなく、広告配信の最適化のためにのみ使用されます。

使用していないサービス

本アプリは以下のサービスを使用していません。

  • Firebase Crashlytics（クラッシュレポート）
  • Firebase Analytics の独立した分析機能（AdMob に付随するものを除く）
  • その他のサードパーティ分析ツール

収集しない情報

本アプリは以下の情報を当社として一切収集しません。

  • メールアドレス、電話番号、氏名等の個人を特定できる情報
  • 位置情報
  • デバイスの連絡先、写真、カメラ等へのアクセス

※ 上記「広告について」に記載の通り、AdMob による広告識別子（IDFA）の取得はユーザーの許可に基づいて行われます。

3. 情報の利用目的

収集した情報は、以下の目的のみに使用します。

  1. ゲームサービスの提供・運営（対戦マッチング、スコア管理、ランキング表示）
  2. 広告の配信
  3. サービスの改善・不具合の修正
  4. 不正行為の検出・防止

4. 情報の保管

  • 保管場所: Google Firebase（Cloud Firestore および Realtime Database）上に保管されます。データはGoogleのセキュリティ基準に基づいて保護されています。
  • 保管期間: ユーザーがアカウントの削除を依頼するまで保管されます。削除依頼後は速やかにすべてのデータが削除されます。

5. 第三者への提供

当社は、以下の場合を除き、ユーザーの情報を第三者に提供しません。

  • 広告配信・効果測定のためにGoogle（AdMob / Firebase Analytics）へ提供する場合（上記「広告について」参照）
  • 法令に基づき開示が求められた場合
  • ユーザーの生命、身体または財産の保護のために必要がある場合

6. ユーザーの権利

追跡の許可・拒否

iOSユーザーは、アプリ初回起動時に表示される「App Tracking Transparency」ダイアログで、広告目的の追跡を許可または拒否できます。この設定はiOSの「設定」アプリからいつでも変更できます。

アカウントの削除

アカウントおよびすべての関連データの削除を希望される場合は、下記のお問い合わせ先までご連絡ください。削除されたデータは復元できません。

データのエクスポート

ご自身のデータの開示を希望される場合は、下記のお問い合わせ先までご連絡ください。

7. 児童のプライバシー

本アプリは13歳未満の児童を対象としたものではありません。13歳未満の児童の個人情報を意図的に収集することはありません。13歳未満のお子様が本アプリを利用する場合は、保護者の監督のもとで行ってください。

8. セキュリティ

当社は、Firebase Security Rulesによるアクセス制御、データの暗号化（通信時のTLS暗号化を含む）、匿名認証によるプライバシー保護など、適切なセキュリティ対策を講じています。

9. プライバシーポリシーの変更

当社は、本プライバシーポリシーを随時更新する場合があります。重要な変更がある場合は、本ページ上で通知します。最新のポリシーは常にこのページでご確認いただけます。
